Replace the default JupyterLab logo in the main area with your own custom logo. Brand your JupyterLab instance for your team, organisation, or project.

## Features

- **Custom main area logo** - Replace the default JupyterLab logo with any image
- **Simple configuration** - Point to your logo file and you're done
- **Zero overhead** - Frontend-only extension, no server component required

## Requirements

- JupyterLab >= 4.0.0

## Install

```bash
pip install jupyterlab_custom_main_logo_extension
```

## Uninstall

```bash
pip uninstall jupyterlab_custom_main_logo_extension
```
